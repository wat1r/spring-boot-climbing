## GitBook教程

```shell
node -v
cnpm install gitbook-cli -g

# 初始化过程比较慢
gitbook -V
CLI version: 2.3.2
Installing GitBook 3.2.3

# 启动服务(到gitbook的项目目录下)
$ gitbook serve

# Serving book on http://localhost:4000

$ SUMMRY下写好目录，gitbook init 生成文件，图片注意相对路径

```
